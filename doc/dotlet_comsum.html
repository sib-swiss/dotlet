Dotlet Command summary

This page presents a summary of dotlet's keyboard and mouse commands (at least those that aren't obvious :-). For detailed help on using dotlet, see the help page.

Mouse commands

  • The lower and upper score thresholds can be moved simultaneously by dragging the mouse in the histogram window. If one threshold is at its maximum and the other st it sminimum (or conversely), this is no longer passible.
  • The dots windows scrolls if you drag the cursor beyond the window boundaries (this happens only if at least one of the sequences is too long for the window, of course).

Keyboard commands

  • Besides the arrow keys, the cursor can be moved using the following keys:
    < up left
    > down right
    ] up right
    [ down left
